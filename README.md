# Python Data & File Handling – Day 2

## Overview

This repository contains my Day 2 Data Analyst Training assignment focused on Python programming, file handling, and data cleaning. The project demonstrates the use of core Python concepts to process and clean CSV data without using external libraries such as Pandas.

## Key Concepts Covered

* List Comprehensions
* Dictionary Comprehensions
* Lambda Functions
* map() and filter()
* File Handling (CSV & TXT)
* Exception Handling (try/except/finally)
* CSV Data Cleaning

## Main Task

A messy CSV file is cleaned using Python by:

* Reading data line by line
* Removing extra spaces
* Converting text values to lowercase
* Removing blank or incomplete records
* Writing cleaned data to a new CSV file

## Additional Exercises

* Extracting even numbers using list comprehension
* Creating number-square mappings using dictionary comprehension
* Sorting tuples using lambda functions
* Converting strings to uppercase using map()
* Filtering words based on length using filter()
* Reading a text file and counting lines and words
* Handling divide-by-zero and invalid input exceptions
* Filtering CSV records based on column values

## Technologies Used

* Python 3
* CSV Module
* File Handling
* Exception Handling

## Learning Outcomes

This project helped strengthen my understanding of Python fundamentals, data cleaning techniques, file processing, functional programming concepts, and error handling practices commonly used in data analysis workflows.
